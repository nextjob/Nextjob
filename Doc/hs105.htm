Order Entry

Selection of the Order Entry menu item results in the Order Entry Window being opened.



The Order Entry Module's form contains tabs for the following functions:

Order - This tab's function is for initial order creation and review including customer shipping and billing information.
 
Item Detail - This tab's function is for order item detail entry and review including customer part number, description, delivery schedule, pricing, and process description.

Extra Charges - This tab's function is for extra charges entry and review.
