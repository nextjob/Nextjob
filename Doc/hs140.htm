Extra Charges



This tab's function is for extra charges entry and review.

Field Descriptions:

EC Code - User selects the desired Extra Charge Code from the drop down list.  Extra Charge Codes are maintained via System Maintenance - Extra Charge Codes.

EC Description - The Extra Charge Code description is displayed here. The default value may be overwritten by the user.

EC Amount -  Users enters the Extra Charge Amount in this cell.

Sales Id - Sales record Id for Extra Charge, populated by the system.

Invoice Number - Invoice Number populated by the system via Invoicing.

Invoice Date -Invoice Date populated by the system via Invoicing.
