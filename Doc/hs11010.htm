User Names

User Names:  QM uses Windows UserNames and Passwords.  This means that in order for non Administrator users to login they must be explicitly allowed access to QMSYS and what ever directory (QMACCOUNTS) the user accounts are stored in.

This application also uses the QM Security feature, see QM Security.



Note Users(Nextjob_PC\Users) is allows access to QMSYS.

If the security tab is not displayed in the properties box for a selected directory or file, check that "Explore > Tools > Folder Options >  View > Use Simple file sharing is NOT checked.
