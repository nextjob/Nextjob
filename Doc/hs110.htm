Order



This tab's function is for initial order creation and review. It includes customer shipping and billing information.

Field Description:

Order #  - Displays the numeric order number, also allows user to select an order for display by entering an existing order number.  Before the system displays the entered order number the routine will warn the user to save changes to currently displayed order (if any).

Order Date - On new order creation user enters the order date (field will default to the current date).

Customer PO - field for the Customer Purchase Order number.

Bill To - User enters know Customer Number or selects Customer Number from the drop down list of existing customers. Field(s) displayed are:  Customer Name , Street Address 1, Street Address 2, City, State Code  and the Bill To Zip Code. Customer Bill To Fields are not editable on an order basis.  A Customer's Bill To information is changed  via the Bill To Edit Button.  Changes to a Customer's Bill To information results in the Bill To information for  ALL orders with the corresponding Customer Number being changed.  Bill To information for new customers can be entered by selecting the Bill To New button, which opens the Customer Entry Dialog. 

Ship To - User enters know Customer Ship To Number or selects Customer Ship To Number from the drop down list of existing customer ship to addresses. Field(s) displayed are:  Customer Name , Street Address 1, Street Address 2, City, State Code  and the Ship To Zip Code. Customer Ship To Fields are editable on an order basis.  Changes to an order's Ship To are effective for that order only. New Ship To information for a customer can be entered by selecting the Ship To New button, which opens the Customer Ship To Entry Dialog. 

Buyer - User enters Buyer's name, selects Buy's name from the drop down list, or accepts the default Buyer for the Customer.

Sales Person - User enters Sales Person's name, selects from the drop down list, or accepts the default Sales Person for the Customer.

Ship Via - User  enters Ship Via method or selects from the drop down list, or accepts the default shipment method for the Customer.

Terms  - User selects the Payment Terms from the drop down list, or accepts default  payment terms for the Customer.

Order Type - User selects the Order Type from the drop down list, or accepts default Order Type.

Order Class - User selects the Order Class from the drop down list, or accepts default Order Class.

Order Notes - User enters any internal Order Notes in this text box.

Buttons:

New - User selects to begin the creation of a new order record.

Find - User selects to open the Order Search Dialog.

File - User selects to save currently active order record.

Cancel - User selects to cancel edits made to active order record.

Delete - User selects to delete active order record (note delete will not be allowed if order is already shipped).
