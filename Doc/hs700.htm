Part Maintenance



This Module's function is to allow the entry and edit of part data.

Part Detail Frame:

Field Descriptions:

Part Number - Used to display or enter the part number. 

Rev - Used to display or enter the part number revision.

Description -  Used to display or enter the part description.

Drawing - Used to display or enter the part drawing number.

SPC - Sales Product Code - Drop down list to select Sales Product Code for part number.

Item Notes -  User entered internal item notes for part number (these notes are not printed on shipping documents).

Material - Used to display or enter the material type for the part.

Pricing Grid:

Quote Qty - Quantity for which price each is valid.

Price Ea.  - Price customer is charged for each item shipped, or a Lot Charge if so specified. Note: price information will not be visible to users with insufficient privileges. 

Date Quoted - Date quote was created.

Notes - Quote notes.

Path - Location of optional picture of part number.

Buttons:

File - Saves changes to the active part record.

Cancel - Cancel changed to the active part record.

Delete Item - Delete active part record.
