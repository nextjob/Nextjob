Users



The User Maintenance form creates and maintains the System File USER records.  These records define users allowed to use the Nextjob application.

The Logout checkbox forces a disconnect from the QM system when the user closes the main menu.

The Menu Access tab defines sub systems the user is allowed access to (if checked the user has access).

The User Restriction tab defines restrictions with in the sub systems. (If checked the restriction is in place).

The Printer Assignment tab defines the printer unit where the associated output will  be printed.  See the QM reference manual SETPTR command for more information on print units. 

Note about printers:
  All printed forms are currently coded to assume  a printer that understand HP PCL 5
  All printed labels are currently coded to assume a label printer than understands  Datamax-O'Neil Programming Language (DPL)
  
For more information see Printing
