Process Groups



The Process Groups form creates and maintains the PROCESS_GROUPS records.  These records are used to populate the Process Step Id drop down list found in the Process Desciption grid used in both Part Maintenance, and Item Detail, process step creation. Process Groups are made up of a list for Process Steps and are used to simplify the process of creating process descriptions.

Field Description:

Group Id - Drop down list to select Process Group to modify.


Group Description - Enter Goup Description

Process Step Id - Used to define the process step list associated with this Group Id.

Buttons:
Delete - User selects to delete the currently highlighted process step.
Insert Before - User selects to insert a process step before the currently highlighted process step.
Insert After - User selects to insert a process step after the currently highlighted process step.
New - Create a new Process Groups record.
File - Saves changes to the Process Groups record.
Cancel - Cancel changes made to the Process Groups  record.
Delete Item - Delete  the Process Groups  record.
