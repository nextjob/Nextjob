Routing Sequence




Track Employee and Machine - When selected labor tickets are entered for machine time and employee labor time as separate tickets.  This allows separate tacking / costing for machine and employee hours.  With this option selected operation status and quantity complete values are maintained from machine labor tickets.

Concurrent Schd - Only relivent if Track Employee and Machine checked.  If checked the greater of Employee Hours or Machine Hours is scheduled.  If not checked Employee and Machine hours are added together and then scheduled out.
