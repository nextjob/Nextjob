Certifcation / Processes



Processes Tab:

Field Descriptions:

Item Certification / Process - Used to define Certifications or Process Steps followed in creating or processing the Part Number.  User selects from a drop down list of standard certification / process steps or enters a custom one.  Note: Certification / Process steps are originally created with the Part Maintenance Dialog, and saved in the part record.  At order entry the Certification / Process steps are copied to the order record. Changing the Certification / Process step of an order will NOT change the Certification / Process step on the part record!

There are two predefined Certification / Process ids,  CUSTOM and NOTES.  The CUSTOM id is used to create a special Certification / Process unique to the part (not defined in Process Steps).   NOTES  are used to create (add) additional description  to the "previous" Certification / Process and is printed in the same Certification / Process block on the Certification / Process sheet.

Certs - Used to indicate that the Certification / Process Description is included on any certification created when shipping the line item.

Buttons:

Modify - User selects to modify the currently highlighted Certification / Process.

Delete - User selects to delete the currently highlighted Certification / Process.

Insert Before - User selects to insert a process step before the currently highlighted Certification / Process.

Insert After - User selects to insert a process step after the currently highlighted Certification / Process.

Print - Create a print out of Certification / Process sheet for the active part.
