Custom Field Control Record

These records control the customization of  "Custom User Fields"
Custom Field Control Records are found in the SYSTEM_FILE and are prefixed by "CFLD*"

The following example shows the customization of the custom user fields on the Order Entry Screen


 
Note the default caption displayed for control lblOpUser1 is defined as "User 1" 



In the Custom Field Control Record for Form OE - Order Tab (CFLD*ORDER_PARENT) we see field CFLD_LABEL_CTRL_ID has a value set to LBLOPUSER1, the corresponding CFLD_LABEL_CAPTION is set to My Date Field1.  This is the caption that is ultimately displayed when the Order Entry Screen is displayed.

Furthermore, corresponding field  CFLD_TXTBOX_CNTRL_ID has a value of TXTOPUSER1 and the corresponding CFLD_TXTBOX_DATA_TYPE has a value of GDDATE specifing that this control's data must be a date. The CFLD_TXTBOX_CONV_CODE field has a value of D2/ specifing the input / output conversion that takes place on data entered into this control. 

Consult the Accuterm 7 GUI manual for valid control data types.
Consult the QM reference manual for conversion codes.
