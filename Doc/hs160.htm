Order Processing Reports

The Reports menu item allows the user to create the following predefined reports:



Open Orders by Order Number - Report of customer orders not shipped sorted by order number.

Open Orders by Customer - Report of customer orders not shipped sorted by customer number.

Orders Shipped - Report of orders shipped  within a date range.

Order History - Report of all orders with order date within a date range.



On selection of a report, a dialog is presented with selection options as defined for the report.
From this dialog the report can be printed or a spreadsheet report (csv file) is created.
If sent to a CSV file (Spreadsheet Report) the csv file will be sent to the users Documents folder, 
  and the program associated with a CSV file will be executed.
