Invoicing

The Invoicing Module is entered by selecting Invoicing from the menu bar.



Once selected the Invoicing Window is displayed.




Module's function is to create an invoice for selected shipped orders.

Create List - User chooses invoicing list creation criteria, Select only items shipped complete or Select all items shipped (item may not be shipped complete).

Create Invoicing list button - User selects to create a list of orders with order items meeting the create list selection  criteria.  The selected orders appear in the "Select Order to Invoice list"

Select Order to Invoice List - User selects order to invoice, "Select Items to Invoice list" is populated with the order's items that have been shipped but not invoiced.

Bill To - Displays Customer Number,  Customer Name , Street Address 1, Street Address 2, City, State Code  and the Bill To Zip Code. Customer Bill To Fields are not editable from this screen, and must be changed from Order Entry.

Select Items to Invoice List - User checks which order item to invoice.

Invoice Total - Used to display total value of items for order selected to invoice.

Command Buttons:

Create List - See above.

Edit Order - Used to open Order Entry for the currently selected order to invoice.  Allows user to correct pricing or other editable fields prior to invoicing.

Cancel - Cancel invoice action on this order.

Invoice - Creates  and prints customer invoice for selected order items.
