Printing

Note about printers:
  All printed forms are currently coded to assume a printer that understand HP PCL 5
  All printed labels are currently coded to assume a label printer than understands Datamax-O'Neil Programming Language (DPL)
  

Printing routines (and calling routine):
APD.FORM   (APDEBIT.DIALOG)
CREDIT.FORM    (AR.REPRINT.DIALOG CREDIT.DIALOG INV.DIALOG)
CUSTOM.PROCESS.FORM  (OE.DIALOG SF.ROUTING.REPRINT)
CERT.FORM    (OE.DIALOG REPRINT.FORMS SHIP.DIALOG)
INVOICE.FORM  (AR.REPRINT.DIALOG INV.DIALOG REPRINT.FORMS)
LOT.TAG.PRINT  (POR.DIALOG)
PACKSLIP.FORM   (OE.DIALOG REPRINT.FORMS SHIP.DIALOG)
PO.FORM   (POE.DIALOG)
PRINT.AP.CHECK
PROCESS.FORM  (OE.DIALOG PN.DIALOG SF.ROUTING.REPRINT)

All assume a printer that understand HP PCL 5

The current label printing routines (and calling routine):
SHIPPING.LABEL (OE.DIALOG POE.DIALOG)
ORDER.LBL  (OE.DIALOG)
RECEIVING.LBL (OE.DIALOG)
PROCESS.LBL   (OE.DIALOG)
All assume connection to a label printer than understands  Datamax-O'Neil Programming Language (DPL)

For label print testing without label printer, create a file to print to (see QM Reference manual)
 CREATE.FILE LABEL_DUMP DIRECTORY PATHNAME C:\QMACCOUNTS\NXT_TEMP
 SETPTR 99,80,66,0,0,3,AS FILE LABEL_DUMP,BRIEF
