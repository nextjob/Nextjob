Routings



Routings Tab:

Routings are used to define the manufacturing steps (sequences) necessay to create a part.  A part may have mulitiple routings identified by the engineering id (Eng Id).

Eng Id - Drop down list of currently defined routings for the active part.

Default End Id - Default routing to use when releasing work orders for the actice part.

Rt Revision Cntl - Routing Revision Control - accesses the revision record for the currently selected part routing.

Buttons:

New Routing -

Copy Routing -

Renumber -

BOM

Print Routing - Create a print out of the Engineering Routing for the active part.

Seq Start / Inc - Starting sequence number and sequence increment used when creating the routing.

Modify - User selects to modify the currently highlighted Routing Sequence.

Delete - User selects to delete the currently highlighted Routing Sequence.

Insert Before - User selects to insert a process step before the currently highlighted Routing Sequence.

Insert After - User selects to insert a process step after the currently highlighted Routing Sequence.

Sequence Grid:

The Sequence Grid displays the individual manufacturing steps (routing sequences) followd to create the part. 

Seq Nbr - Sequence Number

Wc Id - Work Center Id 

Seq Description - Sequence Description
