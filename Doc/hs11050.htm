QM Login Script

The VOC QM system file contains a record called LOGIN.  The record contains a list of statements (script) that run automatically when the user enters the QM system via the login process.  Typically the system administrator would use this record to define the terminal type,  define printers and set other QM system characteristics.   See the QM Reference Manual for further information.  


PA
TERM VT100-AT
pterm case noinvert
IF @TTY = 'vbsrvr' THEN STOP

SETPTR 0,128,66,3,3,1,AT "HP1518", PCL, CPI 16, PAPER.SIZE LETTER, BRIEF
IF @SYSTEM.RETURN.CODE < 0 THEN NXT.MSG.BOX /NextJob/Print Unit 0 Not Assigned "HP LaserJet 5N"/0

SETPTR 1,128,66,3,3,1,AT "HP1518", PCL, CPI 16, PAPER.SIZE LETTER, BRIEF
IF @SYSTEM.RETURN.CODE < 0 THEN NXT.MSG.BOX /NextJob/Print Unit 1 Not Assigned "HP LaserJet 5N"/0

SETPTR 3,128,66,3,3,1,AT "HP1518", PCL, CPI 16, PAPER.SIZE LETTER, BRIEF
IF @SYSTEM.RETURN.CODE  < 0 THEN NXT.MSG.BOX /NextJob/Print Unit 3 Not Assigned "HP LaserJet 5N"/0

SETPTR 4,128,66,3,3,1,AT "DATAMAX", NFMT, BRIEF
IF @SYSTEM.RETURN.CODE  < 0 THEN NXT.MSG.BOX /NextJob/Print Unit 4 Not Assigned "DATAMAX"/0
