Process Steps



The Process Step Maintenance form creates and maintains the PROCESS_STEPS records.  These records are used to populate the Process Step Id drop down list found in the Process Desciption grid used in both Part Maintenance, and Item Detail, process step creation.

Field Description:

Step Id - Drop down list to select Process Step to modify.

Step Name - Enter Step Name

Step Description - Enter Step Description

Process Router Check Boxes - If checked the colomn for the associated value will require entry on the printed process router.

 Time

 Temp

 Volts

 Qty
 
 Oper

 Date
 

Cert Code and Description - A drop down list used to select the Certification Code that goes along with this Process Step.

Work Center - Not implemented  at this time

Hours to Setup - Not implemented  at this time

Hours per Part - Not implemented  at this time

Buttons:
New - Create a new Process Step record.
File - Saves changes to the Process Step record.
Cancel - Cancel changes made to the Process Step record.
Delete Item - Delete  the Process Step record.

There are two special predefined process steps:

NOTES - This process step id is used to add additional process information text to the PRIOR process step created in the Process Description grid of the Part Maintenance, and Item Detail modules.

CUSTOM - This process step id is used to create a custom process step. Custom process steps have unique description text created in the Process Description grid of the Part Maintenance, and Item Detail modules.
