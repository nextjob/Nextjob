System Maintenance



This menu contains buttons for system maintenance functions.

Notes:

Admin Frame buttons are only active for users with Administrator Rights as defined by the QM command ADMIN.USER see QM Security for additional details.  


A.R. Terms - Creates and maintains the A.R. Terms Codes used in Order Entry.

Cert Codes - Creates and maintains Certification Codes.

Countrys - Creates and maintains Country Code list used in address drop down selection lists.

Customer Status - Creates and maintains Customer Status Codes used in Order Entry.

Extra Charge Codes - Creates and maintains the Extra Charge Codes used in invoice.

Lot Charges - Creates and maintains Lot Charge Codes and Lot Charge Amounts.

Material Types - Creates and maintains Material Type Codes.

Order Class - Creates and maintains Order Class Codes used in Order Entry.

Order Type - Creates and maintains Order Type Codes used in Order Entry.

Process Steps - Creates and maintains the standard process steps used in Order Entry and Part Maintenance.

Process Groups - Creates and maintains Process Groups used in Order Entry and Part Maintenance.

Sales Codes - Creates and maintains Sales Prodcut Codes used in Order Entry and Part Maintenance.

Sales Persons - Creates and maintains Sales Person list used in Order Entry.

Ship Via - Creates and maintains Ship Via Codes used in Order Entry.

States - Creates and maintains State Code list used in address drop down selection lists.

Company - Creates and maintains Company Record.

System Defaults - Creates and maintains System Defaults.

Users - Creates and maintains User Control Records.
