Shipping

The Shipping Module is entered by selecting Shipping from the menu bar.



Once selected the Shipping Window is displayed.



This Module's function is to create shipping documents for customer items including pack slips and certifications

Field Description:

Order #  - Allows the user to select an order for shipping by entering an existing order number. 

Customer - Displays Customer Name of currently active sales order (view only).

Cust PO - Displays Customer Purchase Order Number for sales order (view only).

Item Selection List (Grid) - Used to select the line item to ship.  User clicks the up or down scroll arrows until the item number is displayed which they wish to ship, followed by clicking on  any cell within the row.  If the order has only one item, the item is automatically selected.

Part Number - Displays the part number for selected  line item (view only).
 
Description -  Used to display or enter the part description.

Drawing - Used to display or enter part drawing Number.

Balance Due - Used to display balance quantity due for order item (view only).

Qty To Ship - Allows user to enter quantity to ship to customer for currently active order / item.

Packslip Notes - Notes entered at time of shipment to be included on packslip.

Command Buttons:

Cancel - Cancel Order / Item shipping action.

Ship Item - Process Order / Item shipping action.

Cancel Previous Shipment - Opens Cancel Previous Shipment Dialog which allows user to select and cancel a non-invoiced shipment for the active order.
