Part Search Dialog

The Part Search Dialog is any where a part number entry field exists and the entered part number is  not found.



The Part Search Dialog allows the user to search for a part by part number, part description and part drawing.



The user enters the text to search for in the desired text box and clicks the search button.



The system returns the matching items in the corresponding drop down list.



The user selects the part number desired.



The system populates the field that originated the search and processing continues.
