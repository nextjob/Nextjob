Validation File


The Validation_File is a file of records which define various codes and validation data for Nextjob Modules.  The validation file records are maintained by a generic dialog (VF_MAIN) and code (VF.DIALOG). The generic dialog and code is driven by a corresponding record in the file Validation_File_Desc.

  Id = Code Name 
  Field 1 = VM list of code names (CODE ID)
  Field 2 = VM list of code descriptions (DESCRIPTION)
  Field 3 = VM list of code data 1 VALUE 1
  Field 4 = VM list of code data 2 VALUE 2
