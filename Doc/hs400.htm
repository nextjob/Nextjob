Accounts Receivable



Module's function is to process customer checks and apply against customer invoices.

Customer Name - User selects customer name of accounts receivable check from the drop down list.

Check Nbr - User enters customer check number.

Check Amount - User enters customer check amount.

Invoice Grid - User enters Invoice Number to apply check to, system populates Due Date, Invoice Amount, Balance Due and Amount to Apply. User adds optional Write Off Amount and system updates Balance Due and Amount to Apply.

Balance - System displays remaining balance of check amount not applied.

Command Buttons:

Cancel - Cancel customer check application to invoices.

Process - Applies customer check as payment to selected invoices.
